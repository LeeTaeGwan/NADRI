## 📌 About
코로나 때문에 집에만 오래 머물렀던 분들, 갑갑한 일상 속에서 잠깐의 휴식을 찾는 분들을 위한 서비스입니다.
- 동네 사람들이 추천하는 주변의 숨겨진 휴식처를 찾아볼 수 있습니다.
- 나만 아는 숨겨진 휴식처를 사진과 함께 저장하고 공유할 수 있습니다.
